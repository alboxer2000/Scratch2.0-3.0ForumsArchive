    ❓ QNA:

    ❓ Q: Why doesn't my backpack load when I put stuff inside of it?
    ❗️ A: The backpack has a file limit, try deleting things.

    ❓Q: Why doesn't my thing save in the editor?
    ❗️ A: The project.json file cannot exceed 5 MB, this means the code, info about sprites and assets, and any list values, no asset (sound, costumes) cannot exceed 10 MB.

    ❓ Q: How do I report bugs?
    ❗️ A: A bug description, when it started, where it is, a link (if it's in a specific project, please link to it), reproduction steps, expected result, actual result, operating system, browser and Flash version (should be already there. if not, click the globe icon in the tools bar above the topic itself), screenshot.

    ❓ Q: How do I find bugs?
    ❗️ A: https://scratch.mit.edu/discuss/topic/747819/
